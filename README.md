## Project 1: Exploratory data analysis of UA Passengers 1995-2020

### Description
This analysis provides a review of the changes in passenger transportation trends over a 25-year period. By examining various factors such as shifts in passangers preferences, advancements in transportation technologies, and changes in socio-economic conditions, this study aims to identify key patterns and insights that have shaped the passenger transportation landscape.
